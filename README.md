# TestData Manager

A fully-offline web app for tracking tractor R&D test applications — planned vs actual run hours/cycles, failure history, and programme dashboards. Installs as a real app on Android (Edge/Chrome) and iPhone (Safari) via Add to Home Screen. No internet required after install.

## Install on Android (Edge)

1. Open the GitHub Pages URL in Microsoft Edge.
2. Tap the ⋯ menu → **Add to phone** or **Install app**.
3. Open from your home screen — fullscreen, offline, system back button closes overlays.

## Install on iPhone (Safari)

1. Open the URL in Safari.
2. Tap **Share** → **Add to Home Screen** → **Add**.

## Backups

Data lives in localStorage on the device. Use the ⋯ menu → **Export JSON** monthly to save snapshots.
